# Controle de Estoque na FabLab

# Descrição do projeto
Estamos fazendo uma organização para que o itens utilizados no estoque do FabLab sejam utilizados de maneira competente. Visando a pontualidade do usuário ao pegar os itens desejados, atualizar quando o item estiver ou não no estoque, mostrando sua categoria, nome, quantidade e uma agenda de utilização dos itens no estoque.

# Funcionalidade
O **Usuário** terá seu cadastro e com isso poderá observar a finalidade do **Item** e ver a quantidade que tem no estoque do FabLab, e quando necessário, poderá ter uma **Agenda** que o **Usuário** tera a possibilidade de anotar quando for necessário a **Compra** de um **Item** que está em falta no estoque do FabLab. Tendo que o **Item** está relacionado com as outras classes sendo a ideia principal do projeto por ser o estoque de todos os outros itens estarem salvos no banco de dados do FabLab.
